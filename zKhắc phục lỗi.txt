Lỗi Icon component của native-base
link: https://github.com/oblador/react-native-vector-icons
ở phần Option: with CocoaPods

Thực hiện:
- Trong file Podfile thêm đoạn code sau ở phía trên đầu mà dưới require_relative:
pod 'RNVectorIcons', :path => '../node_modules/react-native-vector-icons'

=> cd vào folder ios => gõ lệnh: pod update => Enter để cập nhật lại Podfile

Tiếp tục, trong file Info.plist (bên trong: ios / RN_CLI_Project)
paste đoạn code bên dưới vào trong plist / dict
